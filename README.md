# Windows Systems Administration Portfolio

## Project Overview
This repository serves as a centralized, hands-on portfolio demonstrating core Windows Server engineering, systems administration, and enterprise infrastructure management. The labs contained within this project simulate real-world IT environments, focusing on deployment, security, network services, and automation.

## Technical Skills Demonstrated
* **Hypervisor Management:** Oracle VirtualBox deployment and resource allocation.
* **Operating System Deployment:** Windows Server 2022 installation and initial provisioning.
* **Infrastructure Services:** Hostname management and environment planning.
